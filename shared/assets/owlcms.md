# OWLCMS

OWLCMS is the main competition management system.  It provides
- Automatic recalculation of the lifting order and allowed timing
- All the necessary screens for data entry by the marshal, announcer, secretary
- All the mandated IWF scoreboards and additional variations such as lifting order
- Support for USB/Bluetooth/Joystick refereeing keypads, mobile devices, and dedicated MQTT devices
