# owlcms-firmata

Note. This is an optional module.

##### 

You need to run this module if you have built your own MQTT refereeing devices
using the plans provided in the [owlcms-firmata](https://github.com/owlcms/owlcms-firmata?tab=readme-ov-file#owlcms-firmata) repository.

#####

The software relays the information to and from the devices to the OWLCMS server.
The software only needs to be installed on computers where the devices are connected
- the announcer/timekeeper
- the refereeing laptop
- the jury laptop
